Data Analysis Through Visualization

The intent of the project was for the student to demonstrate the integration of several technologies that have been explored over the span of several months.

We had the opportunity to choose between a provided dataset, or create our own. I opted to create my own data, and selected 627 random cities by making an API call to Open Weather Map. From the initial dataset, I ended up dropping 80 cities due to incomplete data returned from Open Weater Map API. The final data set has 547 cities.

The intent of this activity was to determine whether or not latitude has any impact on humidity, cloudiness, wind speed and temperature. We used Matplotlib to preset the Scatterplots of the values. The following charts is a consolidation of data from these 547 cities around the world. Be sure to check out the Data page to see which cities were included!
